# 🛡️ CyberShield

CyberShield est une application web éducative en HTML, CSS et JavaScript. Elle regroupe plusieurs modules pour découvrir la cybersécurité de façon simple et interactive: tableau de bord, analyse de mot de passe, chiffrement de César, détection de phishing, quiz et actualités.

## Fonctionnalités

- Tableau de bord centralisé avec statistiques
- Analyseur de robustesse de mot de passe
- Chiffrement et déchiffrement de César
- Détecteur de phishing
- Quiz interactif
- Page d’actualités cybersécurité

## Prérequis

- Un navigateur moderne: Chrome, Firefox, Safari ou Edge
- Un éditeur de code ou un terminal si vous souhaitez lancer le projet localement
- Aucun package à installer pour consulter le projet

## Lancer le projet

### Méthode la plus simple

Ouvrez le dossier `cybershield`, puis lancez `index.html` dans votre navigateur.

### Depuis VS Code

1. Ouvrez le dossier du projet dans VS Code.
2. Ouvrez `index.html`.
3. Utilisez l’extension Live Server si vous souhaitez servir le projet en local.

### Depuis le terminal

Commande exacte pour ouvrir `index.html` dans le navigateur (macOS):

Depuis la racine du dépôt:

```bash
cd "Projet-JS---CyberShield-/cybershield" && open index.html
```

Ou, si vous êtes déjà dans le dossier `cybershield`:

```bash
open index.html
```

Option serveur local (si nécessaire):

```bash
npx serve .
```

Ensuite, ouvrez l’adresse indiquée dans le terminal.

## Pages disponibles
